# Assets

This directory contains demo images and visual assets for CYBER SH.

## Generating the Demo Image

Run the demo image generator from the repo root:

```bash
python3 scripts/generate_demo.py
```

This will create `assets/demo.png` - a cyberpunk-styled screenshot showing CYBER SH in action with all 5 modes.

The demo image includes:
- Professional badges (Python 3.10+, 100% Offline, No API Keys, Local GGUF, Pure Python)
- Live console simulation showing Agent mode
- Cyberpunk neon color scheme
- Feature showcase
